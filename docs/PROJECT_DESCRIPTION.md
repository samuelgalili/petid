# PetID - תיאור פרויקט מלא
## הפלטפורמה המלאה לבעלי חיות מחמד בישראל

---

## 📖 סקירה כללית

**PetID** היא פלטפורמה דיגיטלית מקיפה המשלבת רשת חברתית, מסחר אלקטרוני, ומערכת ניהול טיפול בחיות מחמד. הפלטפורמה נבנתה עם גישה ייחודית של "אחריות קודם למכירה" - כל החלטה במערכת נבחנת דרך עיני חיית המחמד.

### חזון
> "עולם שבו כל בעל חיית מחמד מרגיש בטוח בהחלטות שלו."

### משימה
להפוך את הטיפול בחיית המחמד מעניין מסובך למשהו פשוט ונעים, תוך ליווי בעלי חיות בכל שלב.

## 🏗️ ארכיטקטורה טכנית

### Frontend
```
React 18 + TypeScript
Vite (Build Tool)
Tailwind CSS (Styling)
shadcn/ui (Component Library)
Framer Motion (Animations)
React Query (State Management)
React Router (Navigation)
```

### Backend
```

├── PostgreSQL Database
├── Row Level Security (RLS)
```

## 🔐 אבטחה

### Row Level Security (RLS)
כל הטבלאות מוגנות עם מדיניות RLS:
- משתמשים רואים רק את הנתונים שלהם
- תוכן ציבורי נגיש לכולם
- עריכה/מחיקה רק לבעלים
- אדמינים עם הרשאות מורחבות

### אימות
- JWT Tokens
- Refresh Tokens
- Session Management
- Rate Limiting

### הצפנה
- HTTPS בכל התקשורת
- הצפנת סיסמאות (bcrypt)
- הצפנת נתונים רגישים
